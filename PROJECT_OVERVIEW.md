# 怖客 (Buke) - 恐怖灵异内容聚合平台

## 项目总览

**怖客**是一款专注于全简中互联网恐怖灵异内容聚合的网页应用，通过爬虫技术收集B站UP主"大佬何金银"的"道听途说"系列视频内容，整合评论区数据，为用户提供精选的恐怖故事和视频资源。

---

## 一、项目架构

### 1.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | Next.js 14 + React 18 + TypeScript + Tailwind CSS | App Router架构，响应式设计 |
| **后端** | Node.js + Express + Python FastAPI | RESTful API服务 |
| **爬虫** | Python + yt-dlp + Puppeteer + Cheerio | B站视频数据采集 |
| **AI/向量** | ChromaDB + Sentence-Transformers | 语义搜索和相似度计算 |
| **数据存储** | JSON文件 + Chroma向量数据库 | 轻量级数据存储方案 |
| **部署** | Vercel + Netlify + Docker | 前端静态托管 + 后端服务 |

## 二、实现功能

### 2.1 核心功能模块

#### 1) 视频浏览系统
- **首页展示**: 至臻系列、热门视频TOP10、本期主题、提及排行榜
- **视频列表页**: 全部视频展示，支持分页（每页12个视频）
- **分类筛选**: 全部、道听途说正集、道听途说特辑、都市传说
- **视频详情**: 封面、标题、播放量、评论数、上传日期

#### 2) 搜索系统（双模式）
- **普通搜索**: 标题、BV号、期数精确匹配
- **AI语义搜索**: 基于Chroma向量数据库的语义相似度搜索
  - 支持自然语言关键词（如"校园灵异"、"医院怪谈"）
  - 基于视频标题和内容的语义相关性排序

#### 3) 智能筛选系统
| 筛选类型 | 选项 |
|---------|------|
| 排序方式 | 智能排序、浏览量最高、评论最多、最新更新 |
| 浏览量 | 不限、1万+、5万+、10万+、50万+ |
| 评论数 | 不限、100+、500+、1000+、5000+ |
| 时长 | 不限、10分钟内、30分钟内、1小时内 |

#### 4) AI功能
- **AI视频总结**: 基于GPT生成视频内容摘要
- **AI语义搜索**: 基于向量嵌入的语义相似度搜索
- **AI算命**: 命理分析功能（规划中）

#### 5) 用户交互
- **一键直达**: 点击跳转到B站视频页面
- **复制BV号**: 快速复制视频BV号
- **响应式设计**: 适配桌面端、平板、移动端

## 四、产品需求文档 (PRD) 摘要

### 4.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|----------|
| 普通用户 | 无需注册 | 浏览内容、搜索内容、观看视频、使用AI功能 |

### 4.2 页面结构

| 页面 | 核心功能 |
|------|---------|
| **首页** | 至臻系列、热门视频TOP10、本期主题、提及排行榜、访客统计 |
| **视频列表页** | 全部视频展示、搜索、分类筛选、分页浏览 |
| **AI语义搜索页** | 自然语言搜索、智能筛选、语义相似度排序 |
| **故事详情页** | 故事内容展示、相关视频推荐（规划中） |
| **AI算命页** | 命理分析（规划中） |

### 4.3 设计风格

- **主色调**: 黑色、红色、紫色，营造恐怖神秘氛围
- **布局风格**: 响应式设计，卡片式布局，深色背景
- **字体**: 无衬线字体，标题使用粗体和渐变效果
- **图标**: 使用与恐怖主题相关的emoji和图标

### 4.4 技术实现要点

#### API接口
| 路径 | 方法 | 功能 |
|------|------|------|
| `/api/v1/stories` | GET | 获取故事列表 |
| `/api/v1/videos/:bvid` | GET | 获取视频详情 |
| `/api/v1/search` | GET | 搜索故事 |
| `/api/search` | GET | AI语义搜索 |
| `/api/similar` | GET | 相似视频推荐 |

