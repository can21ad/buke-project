## 五、视频跳转流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          视频跳转流程                                       │
└─────────────────────────────────────────────────────────────────────────────┘

用户点击"一键直达"按钮
      │
      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                          构建跳转URL                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  数据准备:                                                                   │
│  {                                                                          │
│    bvid: "BV1xx411c7mD",        // B站视频BV号                              │
│    partNumber: 1,               // 分P号 (如果有)                           │
│    startTime: 754               // 故事开始时间 (秒)                        │
│  }                                                                          │
│                                                                             │
│  URL构建规则:                                                                │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  基础URL: https://www.bilibili.com/video/{bvid}                     │   │
│  │                                                                     │   │
│  │  参数:                                                              │   │
│  │  - t={startTime}    时间定位 (秒)                                   │   │
│  │  - p={partNumber}   分P号 (可选,默认为1)                            │   │
│  │                                                                     │   │
│  │  完整URL示例:                                                       │   │
│  │  https://www.bilibili.com/video/BV1xx411c7mD?t=754&p=1             │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                          执行跳转                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  跳转方式:                                                                   │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  方式1: 新窗口打开 (默认)                                           │   │
│  │  window.open(url, '_blank')                                        │   │
│  │                                                                     │   │
│  │  方式2: 当前页面跳转                                                │   │
│  │  window.location.href = url                                        │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  用户设置:                                                                   │
│  - 记住用户偏好 (localStorage)                                              │

```
┌─────────────────────────────────────────────────────────────────────────────┐

用户滚动页面

│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  const observer = new IntersectionObserver(                        │   │
│  │    (entries) => {                                                  │   │
│  │      if (entries[0].isIntersecting) {                              │   │
│  │        loadMore();                                                 │   │
│  │      }                                                             │   │
│  │    },                                                              │   │
│  │    { threshold: 0.1 }                                              │   │
│  │  );                                                                │   │
│  │                                                                     │   │
│  │  observer.observe(sentinelElement);                               │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  哨兵元素位置:                                                               │

      │
      │ 触发加载
      ▼
┌─────────────────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │

│  ├─ 是否正在加载? → 是: 跳过                                                │
│  ├─ 是否已加载全部? → 是: 显示"已加载全部"                                  │
│  └─ 是否有下一页? → 否: 显示"已加载全部"                                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                          发起加载请求                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │

│  - 设置 isLoading = true                                                    │
│  - 显示加载指示器                                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
      │
      ▼

│     stories = [...stories, ...newStories]                                   │
│                                                                             │
│  2. 更新分页信息                                                             │
│     page++                                                                  │

│                                                                             │

│  4. 重置加载状态                                                             │
│     isLoading = false                                                       │
│                                                                             │
