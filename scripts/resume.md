## 项目经历

### 1. 怖客 (Buke) - 恐怖灵异内容聚合平台

**项目时间**: 2025.03 - 至今  
**项目角色**: 产品经理 + 全栈开发工程师  
**项目链接**: [本地部署] | GitHub: [待补充]

**项目描述**:  
一款专注于全简中互联网恐怖灵异内容聚合的网页应用，通过爬虫技术收集B站博主"大佬何金银"《道听途说》系列的评论区数据，筛选用户推荐度高的故事内容进行整合展示。

**核心功能**:  
- 智能爬虫系统: 基于Puppeteer + Cheerio构建，自动采集B站视频评论数据
- AI内容总结: 集成bobapi多模型(Kimi/GLM/MiniMax)，自动生成视频内容摘要
- 访客统计分析: 实时统计网站访问量、用户行为分析
- 智能推荐系统: 基于用户行为的AI内容推荐，避免重复排行榜内容
- 求助评论挖掘: 自动识别并提取"求出处"、"找故事"类用户评论

**技术栈**:  
- 前端: Next.js 14 + React 18 + TypeScript + Tailwind CSS
- 后端: Node.js + Express
- 爬虫: Python + Puppeteer + Cheerio
- 数据库: MySQL 8.0 + Redis + Elasticsearch
- AI: bobapi (Kimi-K2.5, GLM-5, MiniMax-M2.5)
- 部署: Docker + Nginx

**项目成果**:  
- 完成两版网站迭代，集成AI总结、访客统计、智能推荐功能
- 数据采集: 407+个B站视频，完整的评论数据集
- 爬虫系统: 20+个自动化脚本，支持多版本迭代(v3/v31)
- AI工具: 开发视频总结工具(ai_video_summarizer_pro.py)，支持批量处理
